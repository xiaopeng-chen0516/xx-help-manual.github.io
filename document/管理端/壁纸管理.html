# 壁纸管理

# 菜单地址：内容管理 → 壁纸管理

Tips : 新增壁纸前请先完成分类的新增，否则壁纸无法发布，分类新增请阅读 分类管理

# 1：壁纸发布

# 1.1：点击左上角 新增 按钮,弹出壁纸新增弹窗

# 1.2：点击 + 图标，上传壁纸并选择

# 1.3：填写必要信息，点击确定即可完成发布

# 1.4：选项讲解

  • 标题：壁纸的名字，应该设置成与壁纸相关的词条，比如 超好看的动漫壁纸、唯美风景壁纸、精选的一组手机内置壁纸。
  • 类型：竖版既手机壁纸，横版既电脑、平板等横屏设备壁纸，这一选项会直接影响到客户端壁纸筛选结果。
  • 精选：精选 激活时，该壁纸会在客户端精选页展示，建议将优质壁纸，浏览量高，下载量高，稀缺壁纸设为精选壁纸，并打开加锁功能，提高广告的曝光量。
  • 加锁：加锁 激活时，该壁纸会开启观看广告下载模式,建议配合精选一起使用，加锁壁纸过多会影响用户使用体验，建议合理配置。
  • 分类：将同一风格类型壁纸归类，比如风景、动漫、建筑、星空、动物等，在壁纸发布前应先添加此项，具体操作请阅读 分类管理。

# 2：列表查看

# 2.1：壁纸预览

点击预览图列中的壁纸图片，即可查看大屏预览图（压缩后的），如果要查看原图预览，点击操作栏中 详情 在点击预览图片即可查看原图。

# 2.2：数据排序

点击表头中浏览量、收藏量、下载量、发布时间右侧的上下箭头即可按升序或降序对数据进行排序。

# 3：壁纸修改

# 壁纸修改方式有三种

方式一： 单张壁纸快捷修改上架、精选、加锁；点击某张壁纸对应一行的快捷键快速修改，即时生效，绿色状态标识开启，红色状态标识关闭。

方式二： 点击某张壁纸对应一行的操作栏中的编辑，弹出编辑框，可以修改壁纸的标题、类型、精选、加锁、分类、上架状态

方式三： 批量修改，在壁纸列表左侧栏选中要批量修改壁纸，点击批量操作，选择修改类型，点击确认即可修改

# 4：壁纸删除

Tips : 壁纸删除会同步删除 素材管理中的记录以及云存储中的数据，总而言之就是永久删除该壁纸的源文件及相关数据，删除后用户无法在客户端的我的下载、我的收藏中浏览和下载该壁纸。 如果不想展示该壁纸，请下架该壁纸。

方式一： 单张删除，点击某张壁纸对应一行的操作栏中的删除，点击确定即可删除该壁纸
方式二： 批量删除，在壁纸列表左侧栏选中要批量修改壁纸，点击批量操作，选择批量删除，点击确认即可删除。谨慎操作，删除前请再次确认选中的壁纸
