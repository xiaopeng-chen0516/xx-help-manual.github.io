# 搭建指南

# 1.环境准备

  • HbuilderX (opens new window)
  • 微信开发者工具 (opens new window)
  • Nodejs (opens new window)
  • uniCloud主页 (opens new window)
  • XX壁纸下载地址 (opens new window)

# 2.搭建步骤概述

1.检查电脑环境，确保已经成功安装HbuilderX、微信开发者工具、Nodejs。
2.本项目包含客户端、管理端、service端，客户端运行依赖于管理端和service端，所以需要先部署管理端和service端，再部署客户端，否则客户端无法运行。
3.客户端、管理端、service端部署在同一个云服务空间中，客户端只需要绑定管理端项目即可，详细部署步骤请阅读下述文档。

注册并登录uniCloud → 安装HbuilderX → 安装微信开发者工具 → 打开uniapp插件市场下载并打开项目 → 根据下述步骤部署到云服务空间

# 3.管理端搭建及后台部署

1.下载uniapp插件市场中的XX壁纸管理后台项目
下载地址点这里 (opens new window)

2.点击下载插件并导入HbuilderX，允许浏览器运行HbuilderX，HbuilderX弹出新建uni-app项目，检查项目信息OK后，点击创建，等待项目创建完成

3.项目创建完成后，在HbuilderX的项目列表中打开项目，右击 uniCloud文件夹，点击 关联云服务空间或项目

4.关联服务空间

5.新建服务空间
在uniCloud控制台创建新的服务空间

6.新购并开通服务空间
点击 立即购买，在新页面中点击 确认开通

  • 1：推荐使用阿里云服务器
  • 2：推荐使用按量付费，用多少付多少，不会造成资源浪费
  • 3：如果是第一开通云服务空间，需要缴纳200元保证金，并且对阿里云充值

7.等待空间初始化后在HbuilderX空间列表点击刷新，选择刚在创建的空间点击 关联

8.初始化空间数据
