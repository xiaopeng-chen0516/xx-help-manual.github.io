# 系统简介

# 序

XX壁纸是基于成熟框架VK-admin开发，系统运行稳定，安全可靠，系统分为小程序端和PC管理后台，【高级版】小程序端提供独立后台，可对壁纸进行发布、下架，归类，查看运营数据等操作，简单便捷，随时随地掌控壁纸系统运营；小程序接入流量主功能，广告位灵活配置；整套系统运行在uniCloud云服务器，无需繁琐的运营维护，减轻运营者经营压力。

# 1.后台模块介绍

  • 主页仪表盘：了解系统最新经营情况
  • 内容管理：【壁纸管理、分类管理】
  • 运营分析：【访客列表、壁纸访问、下载列表、收藏列表、广告分析】
  • 微信小程序：【客户端配置、轮播管理】
  • 帮助管理：【帮助列表、帮助分类】
  • 用户角色权限：系统内置功能，对管理后台用户、角色、权限进行配置
  • 系统设置：系统内置功能，对系统进行设置，管理文件库等
  • 快捷link：可通过【系统设置-菜单管理】添加快捷link地址

# 2.小程序端介绍

小程序端有3个Tab切换菜单【首页、分类、我的】，以及内置广告位

  • 首页：【轮播图、通告栏、宫格菜单、壁纸列表】
  • 分类：分类列表、分类标签、版式热切换按钮
  • 我的：【用户信息、我的下载、我的收藏、管理后台（高级版专属）、帮助中心、关于我们、系统设置、联系客服】
